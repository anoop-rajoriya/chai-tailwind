DOMWind
How to use Features Playground

Lightweight utility-first CSS engine without the build step

Powered by the Vanilla JS DOMWind engine. Parse utility classes directly in the browser for rapid prototyping and dynamic styling using the "chai-" prefix.

Github Repo

How to use DOMWind

1

Include the Engine

Link the DOMWind script at the bottom of your HTML document. It automatically initializes and scans the DOM on load. 

<script src="domwind.js"></script>
1

Add Utility Classes

Construct your UI using the chai- prefix. The engine will target only these specific classes. 

<div class="chai-flex chai-bg-black">
  Content
</div>
1

Watch it Render

DOMWind instantly calculates the CSS, maps it to inline styles, and cleans up the class list to keep your DOM tidy 

<div style="display: flex; background: black;">
  Content
</div>
